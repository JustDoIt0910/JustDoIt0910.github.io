由于想使用 protobuf 实现一个自己的 c++ RPC framework，可能需要编写 plugin 来完成一些自定义代码的生成，所以了解了一下 c++ plugin 的开发方法。网上这方面的资料几乎为0，只好去翻官方的 reference 了。
